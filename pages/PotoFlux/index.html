PotoFlux


Description

PotoFlux is a little app containing little useless features.
The source code and installer are avaible on here on github.
Note: PotoFlux is only avaible on Windows.

How to download ?

Download summary :
  • Check Java insallation
  • Install / update Java
  • Download PotoFlux
  • Update PotoFlux

Check Java insallation

You need at least Java 17 to run PotoFlux.
To check if java is installed :
  1. Go to your desktop
  2. Right click
  3. Select "Open in terminal"
  4. Write the command "java -version"
If you have an error, it means you havn't installed Java.
If you have a version lower than 17, you need to update Java.
In these cases, go to Install / update Java.
If you have Java 17 or higher, you can Download PotoFlux

Install / update Java

  1. Go to the Adoptium website
  2. Go to your version
  3. Select de JRE (JavaRuntimeEnvironement) if you just want to run java, select JDK (JavaDevelopementKit) if you want to dev on Java
  4. Download the file and install it
Well done ! You can now Download PotoFlux or Update PotoFlux.

Download PotoFlux

  1. Go to the PotoFlux Github release page
  2. Select the release you want (We suggert the lastest, and avoid the beta as they are ofter buggy. Information about the code.)
  3. In the assets, select the installer type you want, depending on your version (ex. msi for windows)
  4. Run the installer. Once done, you can run PotoFlux with the new icon on your desktop !
For any issues, you can opent an issue on the github issues pages.

Update PotoFlux

To update PotoFlux, you first need to uninstall manually the current version.
Then, you can Download PotoFlux again, in the new version.
All you settings and data (Todos...) will not be deleted during the uninstall.
