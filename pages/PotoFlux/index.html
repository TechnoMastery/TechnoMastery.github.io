PotoFlux


Description

PotoFlux is a little app containing little useless features.
It gets it's strength by its mod API, allowing anyone to create a little extension.
The source code and installer are available on here on GitHub.
Note: PotoFlux is only available on Windows.

Information about the code

All the source code is available on our GitHub.
Feel free to check it, re-use it. We will never place malware in the code.
For any more issues or questions, you can open an issue on the GitHub or join our Discord server (French server, english understood).
You can also check the Download Potoflux section to download Potoflux's source code by versions.
