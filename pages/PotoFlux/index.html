Description

PotoFlux is a little app containing little useless features.
It gets it's strength by its mod API, allowing anyone to create a little extension.
The source code and installer are available on here on GitHub.
Note: PotoFlux is only available on Windows. Click here if you'd like to help port to another OS.

Information on the app and the code

All the source code is available on our GitHub.
Feel free to check it, re-use it (remember to respect the license). We will never place malware in the code.
For any more issues or questions, you can open an issue on the GitHub or join our Discord server (French server, english understood).
You can also check the Download Potoflux section to download Potoflux's source code by versions.

Supported OS

The app is actually only available on Windows. If you want to download it without being on Windows, the app needs to be ported.
I actually can't port the app, because I only have access to Windows devices, making debugging impossible.
If you are the owner of a Mac, Linus or other device, and you want to be able to download the app, you can help me port the app by joining my Discord and message me (@minheur2000).
If you already know how to port to your OS, you can also fork the repository, port it by adding your OS installer creation on the build command execution.
