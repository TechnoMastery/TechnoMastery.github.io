Memory game

Test your memory with this fun game! Try to find all the pairs of cards in the shortest time possible.


Memory

Rules

The rules are simple : you have to find the pairs in the shortest time possible.
Click on a card to start. The timer begins when you turn the first card.

When you find a pair, the cards will stay turned. If you don't find a pair, the cards will turn back after 1 second.

When you find all the pairs, the game is over and you can see your time.

Good luck!
